# Product Hunt Description

## Tagline
An institutional-grade, hallucination-free AI Research Desk for Finance.

## Description
Financial analysts spend hours navigating dense 100-page SEBI circulars and AMC factsheets. Standard ChatGPT wrappers fail here because they hallucinate numbers and cannot prove their claims. 

**SIF Copilot** is a high-performance Retrieval-Augmented Generation (RAG) workspace built specifically for Specialized Investment Funds. 

It implements a "Hybrid Imperative" architecture that preserves exact table structures, sanitizes noise, and forces Llama-3.1 (running on blazing-fast Groq LPUs) to output verifiable `[Source N]` citations. 

**Key Features:**
- 🕵️‍♂️ **Evidence Explorer**: Click any citation to instantly view the exact document, page number, and chunk used to generate the answer.
- ⚡ **Groq + Qdrant**: Sub-second retrieval and generation.
- 📊 **Dynamic Trust Metrics**: View the chunk count, search latency, and cosine-similarity confidence score for every response.
- 🎨 **Bloomberg-Meets-Perplexity UI**: A high-density, dark-mode glassmorphism interface built for power users.

Open-source and fully containerized. Stop guessing. Start verifying.
